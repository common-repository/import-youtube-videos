A simple YouTube video importer plugin. Import YouTube videos automatically to your WordPress site.


== Description ==

This is one of the best video importer WordPress plugins which is widely used by others. With the help of this wordpress  plugin, you will be able to add the Youtube videos to your WordPress website with so much ease.  It can automatically create the posts on your WordPress site from Youtube channels, playlists, searches etc. These posts will have some details such as featured image, post content, title & many more. So you’ll exactly showcase your Youtube content with any hindrance.
This Plugin Import video from youtube channel. With this plugin you can import YouTube channels videos to your WordPress website. This helps to easily import YouTube videos into WordPress as Custom posts (videos).
The plugin can import all your YouTube videos or you can select the video by date or you can select video by using the preview button. If you can import without preview then it will import all video from the selected channel. Before you start using this plugin please add google youtube api key in setting otherwise it will not work as shown in screenshot1.
This plugin creates an option under tools in admin name import youtube here you can select your API key and channel.
The plugin is highly configurable and provides you with various options for fetching videos. Whether you want to pull videos from a channel, a playlist, or a specific user and even search query, we got you covered.
Also, there are several smart options related to the import process itself. Choose a post type, post status, automatically add video description into post content, assign categories, tags or any custom taxonomies while importing the videos.
FEATURES
Bulk import YouTube videos
The plugin automatically adds new videos on your website from the YouTube channels, playlists or users that you are following, whenever they upload new videos. Hence, the plugin keeps your website up to date.
There is no restriction on importing large quantities of videos on a WordPress website. So you don’t have to worry about missing any video.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'import-youtube-videos'
3. Activate import-youtube-videos from your Plugins page.

= From WordPress.org =

1. Download import-youtube-videos.
2. Unzip plugin.
2. Upload the 'import-youtube-videos' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate import-youtube-videos from your Plugins page.

= Usage =

1. Go to admin > Tools > Import Youtube to got to settings.
2. Enter Api key
3. Enter channel id and click preview or import.


= Uninstalling: =

1. In the Admin Panel, go to "Plugins" and deactivate the plugin.
2. Go to the "plugins" folder of your WordPress directory and delete the files/folder for this plugin.



== Frequently Asked Questions ==

= About Plugin Support? =

Send your question on info@sivahtech.com and we will try to answer your question as quick as possible.

= The import failed or takes too much time to process? =

You can run the improter multiple times, as it will never import the same post twice. Once all videos are imported, only future posts would be added, assuming you selected the continuous import option.

= What is an "API Key"? =

To import your videos from YouTube, you'll have to create a key and save it in the plugin's settings.An "API Key" is like a password that allows you to search, process and import videos from YouTube.It is required by YouTube/Google that you have a valid key before you can import any data from their platform.

= Do I need to add a YouTube API key? =

Yes, a valid YouTube API key is required, you can create a key on the Google Cloud Platform.

= Is there a limit on the number of imported videos =

Yes. Google limits your API key with a daily quota.If you're trying to import a large channel with thousands of videos, you can ask Google for an increase -[https://support.google.com/youtube/contact/yt_api_form](https://support.google.com/youtube/contact/yt_api_form)

= The import does not work for my YouTube feed =

First, make sure your server is up to date with the requirements - we recommend PHP 7.1 or above.Second, feel free to contact us if you encounter any issues.
